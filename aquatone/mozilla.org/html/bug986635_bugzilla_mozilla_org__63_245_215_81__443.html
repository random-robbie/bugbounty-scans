Currently we have two initial mockups for private browsing dark theme:
http://people.mozilla.org/~shorlander/private-browsing-mode/mockups/australis-pbm.png

We need to finalize the visual design for private browsing mode, including:
- Different platforms
- Tweak the current design so that it's more subtle, less dramatic
- Add an icon indicator in addition to the dark theme, to make it more clear which mode the user is in, especially for people using dark theme 
* The placement/style of the indicator need to be consistent with the multi-user feature that might come out in the future
