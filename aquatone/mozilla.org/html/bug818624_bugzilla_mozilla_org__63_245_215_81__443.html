It would be cool to feature all our contributors in a list somewhere that users can get to. On Firefox desktop, there is a link to about:credits in the about box, so maybe we can make a similar link in our "About Firefox OS" section.

Initially I was thinking we would make a new credits page for Firefox OS, but there would actually be a lot of overlap with about:credits, so maybe we should just use that. After all, Firefox OS was built on top of a lot of work that was already done by the Mozilla community that's credited in about:credits.

We may want to consider adding a link to 'about:credits' on the Device information screen in the Settings app, which I think is the Firefox OS equivalent to 'about:firefox' on desktop. The link could be added as a button in the About Firefox OS section of this screen.

> Please note that the about:credits page is already linked from inside the
> browser application, although I think should be more visible including some
> general information about Mozilla and how newcomers can get involved.

Great points Tomer. For reference, this link can be found on Firefox OS by:
- Opening Firefox
- Tap on the tab number to reveal the Settings icon
- Tap on the Settings icon
- Tap on About Firefox
- Tap on Credits

Note that the Credits page layout could be more responsive on a mobile screen so you don't need to scroll horizontally.
